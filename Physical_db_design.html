1. Physical database design process යනු කුමක්ද?
    2. Answer
    • Physical database design යනු දත්ත ආකෘතියක් විශේෂිත දත්ත සමුදා කළමනාකරණ පද්ධතියක (DBMS) physical data ව්‍යුහය බවට පරිවර්තනය කිරීමේ ක්‍රියාවලියයි.



  2. Logical සහ Physical දත්ත සමුදා නිර්මාණය අතර වෙනස ලියන්න
    3. Answer
      Logical Database Design Physical Database Design
      දත්ත ගබඩාවේ භෞතිකව ක්‍රියාත්මක වන ආකාරය නොසලකා දත්ත විස්තර කරයි. එය සැබෑ දත්ත සමුදාය ගොඩනගා ඇති ආකාරය නිරූපණය කරයි
      දත්ත මූලද්‍රව්‍ය (Data Element) සහ ඒවායේ සම්බන්ධතාවය නිර්වචනය කරයි. Actual දත්ත සමුදාය සංවර්ධනය කිරීම.



  3. Physical Design හි context තුළ Normalization යන්නෙන් අදහස් කරන්නේ කුමක්ද?
    4. Answer
    • Normalization යනු දත්ත සමුදායක් තුළ දත්ත සංවිධානය කිරීමේ ක්‍රියාවලිය වන අතර එමඟින් එය යම් අවශ්‍යතා සපුරාලන අතර එමඟින් දත්ත අතිරික්තය අඩු කිරීමට සහ දත්ත අඛණ්ඩතාව වැඩි දියුණු කිරීමට උපකාරී වේ.
