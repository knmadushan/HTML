1. RDBMS යනු කුමක්ද?
    2. Answer
    • RDBMS යනු Relational Database Management System යන්නයි. RDBMS වගුවේ තීරු අතර පොදු ක්ෂේත්‍ර මගින් සම්බන්ධ වන වගු එකතුවක දත්ත ගබඩා කරයි.

  2. SQL යනු කුමක්ද?
    3. Answer
    • SQL යනු Structured Query Language වන අතර එය දත්ත සමුදාය සමඟ සන්නිවේදනය කිරීමට භාවිතා කරයි.

  3. SELECT ප්‍රකාශය යනු කුමක්ද?
    4. Answer
    • SELECT විධානයක් දත්ත සමුදා වගු එකකින් හෝ වැඩි ගණනකින් පේළි බිංදුවක් හෝ වැඩි ගණනක් ලබා ගනී

  4. Table සහ Field යනු මොනවාද?
    5. Answer
    • වගුවක් යනු සම්බන්ධතා දත්ත ගබඩාවක පේළි සහ තීරු වල සංවිධානය කර ඇති දත්ත සංරචක එකතුවකි. Table සබඳතාවල usefull නිරූපණයක් ලෙසද සැලකිය හැකිය. දත්ත ගබඩා කිරීමේ මූලිකම ආකාරය වගුවයි

  5. SQL උදාහරණයක් යනු කුමක්ද?
    6. Answer
